22. Генерация скобок
Уровень: Средний
Темы:
Иконка премиум-доступа Компании

Дано число n — количество пар скобок. Напишите функцию, которая генерирует все возможные комбинации правильно сформированных скобочных последовательностей.

Пример 1:

Ввод: n = 3
Вывод: ["((()))","(()())","(())()","()(())","()()()"]

Пример 2:

Ввод: n = 1
Вывод: ["()"]

Ограничения:

    1 <= n <= 8
